## 🚀 How to Deploy to Smartphones

### Quick Deploy (3 Steps):

```powershell
# Step 1: Navigate to project
cd c:\ai\school-tts

# Step 2: Run deployment script
.\deploy.ps1

# Step 3: Choose option 1 (Production)
```

### Alternative: Drag & Drop Deploy
1. Go to https://app.netlify.com/drop
2. Drag the `school-tts` folder
3. Get instant URL!

### After Deployment:
1. Copy your deployment URL (e.g., `https://celtech-voice.netlify.app`)
2. Open on smartphone browser
3. Install the app:
   - **Android**: Menu (⋮) → "Install app"
   - **iOS**: Share (□↑) → "Add to Home Screen"
