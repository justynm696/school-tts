# 🚀 Quick Start - Deploy CeltechVoice in 5 Minutes

## Fastest Method: Netlify Drop (1 Minute!)

### Step 1: Prepare Your Files
Your files are ready in: `c:\ai\school-tts`

### Step 2: Deploy
1. Open this link: **https://app.netlify.com/drop**
2. Drag the entire `school-tts` folder onto the page
3. Wait 30 seconds for deployment
4. **Done!** Copy your live URL

### Step 3: Share
Your app is now live! Share the URL:
- 📱 Works on mobile browsers
- 💻 Works on desktop browsers
- 🔒 Automatic HTTPS enabled
- 🌍 Accessible worldwide

---

## Alternative: Run Deployment Script

```powershell
cd c:\ai\school-tts
.\deploy.ps1
```

Choose option 1 for Netlify, option 2 for Vercel, or option 3 for GitHub Pages.

---

## Mobile Installation Instructions

### For Students with Android:
1. Open the deployed URL in Chrome
2. Tap the menu (⋮) → "Add to Home screen"
3. Tap "Add"
4. CeltechVoice icon appears on home screen!

### For Students with iPhone:
1. Open the deployed URL in Safari
2. Tap Share button (□↑)
3. Scroll and tap "Add to Home Screen"
4. Tap "Add"
5. CeltechVoice icon appears on home screen!

## Create QR Code for Easy Sharing

1. Go to: **https://qr-code-generator.com**
2. Enter your deployed URL
3. Download the QR code
4. Print and post around campus!

Students can scan the QR code to instantly access CeltechVoice.
