                        DATA FLOW EXAMPLES

EXAMPLE 1: Search Flow
──────────────────────
User Input: "library"
    ↓
PROCESS 1.0: Search Engine
    ↓ Read from D1 (School Data)
    ↓ Read from D3 (Navigation Data)
    ↓
Results: [Facility: Library, Guide: 2nd Floor, Location: Library Room]
    ↓
Display to User


EXAMPLE 2: TTS Playback Flow
─────────────────────────────
User Clicks: Event Card
    ↓
PROCESS 2.0: Content Display
    ↓ Extract content
    ↓
PROCESS 4.0: TTS Engine
    ↓ Read from D2 (User Preferences)
    ↓ Create utterance
    ↓
Web Speech API
    ↓ Generate audio
    ↓
Audio Output to User


EXAMPLE 3: Voice Command Flow
──────────────────────────────
User Speaks: "Show me the canteen"
    ↓
Web Speech API (Recognition)
    ↓ Convert to text
    ↓
PROCESS 3.0: Voice Assistant
    ↓ Parse command
    ↓ Read from D3 (Navigation Data)
    ↓
PROCESS 5.0: Interactive Maps
    ↓ Display floor plan
    ↓ Highlight location
    ↓
Visual Output to User
