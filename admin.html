ℹ️ About

This admin panel manages the content served by the V.I.R.A. Virtual Interactive Resource Assistant at Celtech College Olongapo.

Changes are saved to localStorage in your browser and picked up automatically by the main V.I.R.A. app when opened in the same browser.
