# How to Install V.I.R.A. on Android 📱

V.I.R.A. (Virtual Interactive Resource Assistant) is a **Progressive Web App (PWA)**, which means you can install it directly from your web browser without going to the App Store.

## 🚀 Fast Installation Guide

### Option 1: Using Google Chrome (Recommended)
1. Open **Chrome** on your Android phone.
2. Navigate to the website URL (where you deployed the app).
3. Wait for the **"Add V.I.R.A. to Home screen"** banner to appear at the bottom.
4. Tap **"Install"**.
5. The app will be added to your app drawer and home screen!

### Option 2: Manual Installation via Menu
If the banner doesn't appear:
1. Open **Chrome** and go to the V.I.R.A. website.
2. Tap the **three dots (⋮)** in the top-right corner.
3. Select **"Install app"** or **"Add to Home screen"**.
4. Tap **"Install"** to confirm.

### Option 3: Using Samsung Internet
1. Open the website in **Samsung Internet**.
2. Tap the **Menu (≡)** icon in the bottom right.
3. Tap **"Add page to"**.
4. Select **"App screen"** or **"Home screen"**.
5. Tap **"Install"**.

## ✨ App Features on Android
Once installed, V.I.R.A. works just like a native app:
- **Offline Access**: Use the app even without internet connection (cached content).
- **Full Screen**: Runs without the browser address bar for a cleaner experience.
- **Fast Loading**: Loads instantly from your phone's storage.
- **App Icon**: Appears with the official Celtech logo in your app list.

## 🛠️ Troubleshooting
- **"Install" option missing?** 
  - Make sure you are using a supported browser (Chrome, Edge, Samsung Internet).
  - Ensure you are visiting the site via HTTPS (secure connection).
  - Try refreshing the page.
